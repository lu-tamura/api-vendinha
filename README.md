## Funcionalidades para o usuário

### Criar Usuário
- **Método**: POST
- **URL**: `/user`
- **Campos**:
    1. `name`
    2. `email`
    3. `password`
    4. `is_active`
    5. `cpf_cnpj`

**Exemplo de `curl`**:

```bash
curl --location 'http://localhost:8000/api/users' \
--header 'Content-Type: application/json' \
--data-raw '{
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123",
    "is_active": true,
    "cpf_cnpj": "12345678901"
}'
```
### Atualizar Usuário
- **Método**: PUT
- **URL**: `/user/{id}`
- **Campos**:
    1. `name`
    2. `email`
    3. `password`
    4. `is_active`
    5. `cpf_cnpj`

**Exemplo de `curl`**:

```bash
curl --location --request PUT 'http://localhost:8000/api/users/1' \
--header 'Content-Type: application/json' \
--data-raw '{
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123",
    "is_active": true,
    "cpf_cnpj": "12345678901"
}'
```
### Encontrar Usuário pelo ID
- **Método**: GET
- **URL**: `/user/{id}`

**Exemplo de `curl`**:

```bash
curl --location 'http://localhost:8000/user/1'
```

### Inativar ou Ativar usuário pelo ID
- **Método**: PUT
- **URL**: `/user/{id}/status/{status}`
- **Campos**:
    1. `is_active`

**Exemplo de `curl`**:

```bash
curl --location --globoff --request POST 'http://localhost:8000/api/users/{id}/status/{is_active}'
```
